![VibeDuel demo](demo.gif)

## Install

```bash
curl -fsSL https://vibeduel.ai/install | bash
```

Open a new terminal, then run:

```bash
vibeduel
```

## How to use it

### Single-model mode
Enter `/models` and pick your favorite model. Each request costs **1 credit**.

### Duel mode

Enter `/models` and pick `duel`. Each vote gives **50 credits** and goes to the leaderboard.
